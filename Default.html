What is Singularity/Apptainer?

"Apptainer/Singularity is the most widely used container system for HPC. It is designed to execute applications at bare-metal performance while being secure, portable, and 100% reproducible."1

Back in November 2021, the Singularity project has forked into a commercial offer with a community version from the Sylabs company, which kept the Singularity name, and a fully open-source project now named Apptainer under the Linux Foundation.
